# 欢迎来到长江组件库
一个由长江前端团队自己打造的库
### 安装组件库

```javascript

npm install ahcj-generals --save

```
### 使用方式

```javascript
// 加载样式
import 'ahcj-generals/dist/index.css'
// 引入组件
import {Button, Panel} from "ahcj-generals"

```
